
# Project_R304

Our repository for the Project_R304, a simulation of the Armorique.
This project if a student creation for the R3.04 resource of the BUT2 - IUT Gaston Berger, Aix-en-Provence.

We did our best for the project and hope you will enjoy it.

## Documentation

In order to build the documentation, you will need to type the following command:
```
foo@bar:project$ javadoc -d Documentation -sourcepath src -subpackages controller:model:view:includes:game
```
